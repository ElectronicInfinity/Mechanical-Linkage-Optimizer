# Mechanical Linkage Optimizer
Optimizes mechanical linkages of the form of a 4-bar linkage with an extension element.
The provided program also contains in it a lot of settings for the optimizer and some experiments.

WORKS BEST WITH 1 OR 2 TARGET POINTS - ACCURACY RAPIDLY DROPS AFTER 2 POINTS DUE TO LACK OF DEGREES OF FREEDOM IN A 4 BAR LINAKGE SYSTEM
